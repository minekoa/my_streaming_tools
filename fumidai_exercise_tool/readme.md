# fumidai_tool.pyw

## これは何？

踏み台昇降配信専用の配信ツール

踏み台昇降配信は、30分1セット x 3回の踏み台昇降を配信する。

私のPCのプアさと、トラッカーを持っていないことから
運動のフルトラ配信が難しいため、
トラッキングはバストアップのみとし、
画面にはわかりやすく踏み台昇降するアニメーションGIFアニメを表示したい。

* 踏み台昇降のタイマー機能
    * 開始、ポーズ ができる
    * 30分で自動フィニッシュ
* タイマーに連動した配信ソースの生成
    * セット数カウントと、30分のカウント表示 (テキストファイル出力）
    * アニメーションGIFの切り替え (ready, exercising, braking)
    * タイマー終了時にチャイム音をならす機能
        * のため、BGMも本ツール起動時に自動再生（ダッキングしたいので）


## もっと詳しく


コード内の`Config` に記されているデフォルト状態での期待されるファイル配置。


```
root/
   +- fumidai_tool.pyw           ..本ツール
   +- sound/
   |  +- 学校のチャイム.mp3
   +- bgm/
   |  +- 早起き体操.mp3
   +- 画面小物/
   |  +- 待ち_1-2_0.5.gif
   |  +- 踏み台昇降_0.2_0.5.gif
   |  +- 息切れ_0.5.gif
   +- tmp/                       ..映像ソースの出力先
      +- 踏み台タイマー.txt
      +- 踏み台昇降配信.gif
```

利用したいファイルに合わせて `Config` データクラスを適当にいじってください。

BGM,サウンドのボリュームはハードコートしてしまってるので変更したい場合はコード内を探してください。
