# roadshow_tool.pyw

## これはなに？

同時視聴配信用のOBSリソース制御ツール。

* カウントダウン＆経過時間タイマー
* タイマーに連動した画像リソースの制御
* タイマーに連動したブザー音などの効果音制御

を行う

## もっと詳しく

以下の制御を行う。

### タイマー表示機能（連動したカウントダウン・経過時間）

タイマーを推して30秒のカウントダウン表示のあと、ストップするまで経過時間を表示


### タイマーと連動した効果音再生

タイマーONをしたタイミングで、開演ブザーをならし、映写機のリール音をカウントダウン中に鳴らす。
（リール音は視聴野妨げになるのでカウントダウン中のみ）

これらの音は標準の音声出力ではなく、特定の音声チャンネルに出力する。

※同時視聴配信のため、標準の音声出力は同時視聴映像の音声が流れており、これを配信に入れられないため。


### タイマーと連動したカウントダウン表示

10秒から1秒までのカウントダウン画像(アニメーションgif)をタイマーと連動してソースに投入する。

### 背景画像の差し替え

タイマーと連動して、同時視聴中（タイマー動作中）は暗い劇場のイメージに、そうでないときは明るい劇場のイメージに差し替える。


### スクリーンイメージの制御

同時視聴中は投影中をイメージした明滅状態に、そうでないときはくっきりとした静止画像に。


ベースのスクリーンイメージとは別に、投影中の明滅アニメーションの上に覆いかぶさる同じ画像のスクリーンイメージ画像を用意し、ソースを差し替えて実現。
（本当は Animation PNG ファイルの差し替えで再現したかったが、OBS側がファイルの差し替えを関ししていない）
